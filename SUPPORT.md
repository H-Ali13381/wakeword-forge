# Support

## Community support

Use GitHub issues for:

- Reproducible bugs.
- Installation problems.
- Documentation gaps.
- Feature requests.
- Questions about supported workflows.

Please include:

- Operating system and Python version.
- Exact command run.
- Full error output.
- Whether you used microphone recording, TTS augmentation, optional datasets, or a custom ONNX runtime.

## Scope

Best-effort public support covers the open-source package and documented examples.

Out of scope for free support:

- Debugging private datasets without a reproducible example.
- Certifying third-party licenses.
- Guaranteeing accuracy for a specific speaker, accent, room, or microphone.
- Maintaining downstream integrations not owned by this project.

Commercial support may be available for custom evaluation, integrations, and deployment tuning.
